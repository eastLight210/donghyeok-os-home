# DonghyeokOS Home

[DonghyeokOS](https://donghyeok.net) is a personal home site built as a warm, desktop-inspired web experience.

## Development

```sh
npm install
npm run dev
```

Before publishing changes, run:

```sh
npm test
npm run lint
npm run build
```

## Deployment

Cloudflare Workers Builds deploys the `main` branch to the `donghyeok-os-home` Worker. Production is served from [donghyeok.net](https://donghyeok.net).
